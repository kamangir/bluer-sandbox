# offline_llm

```bash
@offline_llm \
	install \
	[dryrun]
 . install offline_llm.
@offline_llm \
	model \
	download \
	[dryrun,overwrite,tiny]
 . download the model.
@offline_llm \
	model \
	get \
	[tiny] \
	[filename | object | repo]
 . get things.
@offline_llm \
	prompt \
	[tiny,~upload] \
	"<prompt>" \
	[-|<object-name>]
 . "<prompt>" -> offline_llm.
```
